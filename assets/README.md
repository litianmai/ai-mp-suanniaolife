# 游戏资源文件夹

用于存放小程序的游戏素材资源。

## 目录结构

```
assets/
├── images/          # 图片资源
│   ├── backgrounds/ # 背景图片
│   ├── characters/  # 角色图片
│   └── items/       # 道具图片
├── sprites/         # 精灵图/雪碧图
│   ├── missiles/    # 导弹素材
│   ├── explosions/  # 爆炸效果
│   └── effects/     # 特效素材
├── audio/           # 音频资源（待创建）
│   ├── sfx/         # 音效
│   └── bgm/         # 背景音乐
└── fonts/           # 字体文件（待创建）
```

## 素材来源

- **Kenney.nl**: https://kenney.nl/assets
- **OpenGameArt**: https://opengameart.org/
- **itch.io**: https://itch.io/game-assets

## 许可说明

使用前请确认素材的许可协议：
- CC0: 完全免费，可商用
- CC-BY: 需要署名
- GPL: 开源项目可用
